# Architecture Decision Records

This folder records meaningful product, governance, and architecture decisions for Chirin Ivatan.

ADRs are short notes explaining:

- what was decided
- why it was decided
- what alternatives were considered or rejected
- what consequences the decision creates

Use ADRs for decisions that future contributors, reviewers, maintainers, or capstone panelists may reasonably ask about.
