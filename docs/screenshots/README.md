# Screenshots

Place README screenshots here. The main project README links to these filenames:

- `dictionary.png` — a dictionary entry with meaning, phonetics, and audio
- `folklore.png` — the folklore archive category view
- `reviewer-dashboard.png` — the reviewer dashboard showing a pending submission

## Guidelines

- Capture at a consistent browser width (around 1280px) so the images look uniform.
- Use approved/sample content, not real contributors' personal data.
- Keep each file reasonably small (PNG under ~1 MB) so the repository stays light.
- Use the exact filenames above, or update the links in the main `README.md`.

Once the images are added, uncomment the image links in the project `README.md`.
